BlueNRG-1,2 Development kits (STSW-BLUENRG1-DK)
Release Notes

This document contains the release notes of the BlueNRG-1_2 DK Development kits (STSW-BLUENRG1-DK).

  • BlueNRG-1_2 DK SW package provides the software drivers, library and examples for the BlueNRG-BLE Stack Library (STMicrolectronics BlueNRG-1, BlueNRG-2 Bluetooth Low Energy stack), 2.4 GHz proprietary radio solutions and all the other device peripherals (ADC, GPIO, I2C, MFT timers, RNG, RTC, SPI, UART, WDG). It also includes the BlueNRG-1, BlueNRG-2 Navigator tools and the BlueNRG-X Radio Init Wizard PC application for Windows PC.
  • This document describes the new features, changes, solved issues and limitations associated to each new release of the SW package.
  • NOTE: The code provided by ST, intended to be run on BlueNRG-1,BlueNRG-2, both in source format or in binary format (Bluetooth Low Energy Stack), do not include any part released with open source license.
  • A PC running Windows 10 operating systems (64 bits version) and administrator rights, with available USB ports is required for installing the BlueNRG-1_2 DK SW package.
